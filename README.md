# üîß Installation
- Install the Weeaboooify app
- Open LSPosed and enable Weeabooify
- Check SystemUI and Framweork
- Go back to module list and long press at Weeabooify then tap Re-optimize
- Open Weeaboify grant all permissions then login to your account
- Wait for installation finish then tap on reboot and enjoy!
> [!Note]
>
> KSU users have to manually enable root access from the KernelSU app itself.

# ü§î FAQ
<details>
  <summary>Is my device supported?</summary>

  - Weeabooify only supports for Pixel or AOSP custom rom based, not support for OEM ROMs like MIUI, OneUI, OxygenOS, ColorOS, iOS, etc.
</details>

<details>
  <summary>How to fix bootloop/bsod?</summary>

  - Enter safe mode or delete Weeabooify module in data/adb/modules/ using custom recovery.
</details>
